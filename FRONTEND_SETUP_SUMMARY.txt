================================================================================
                         TROUBLESHOOTING
================================================================================

If frontend won't start:
  1. Check port 3000: netstat -ano | grep 3000
  2. Kill if needed: taskkill /PID <PID> /F
  3. Try different port: PORT=3001 pnpm dev

If can't reach API:
  1. Check Gateway: curl http://localhost:8080/api/v1/health
  2. Verify .env.local: cat .env.local
  3. Check Network tab in browser DevTools

If build fails:
  1. Clear cache: rm -rf .next
  2. Reinstall: pnpm install
  3. Rebuild: pnpm build
  4. Check types: pnpm type-check
