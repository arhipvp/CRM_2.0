KEY PATTERNS & PRACTICES
=========================
1. Immutable State Updates
   setDeals(prev => prev.map(d => d.id === id ? {...d, field: val} : d))

2. Modal Type Union
   type Modal = 'addDeal' | 'addClient' | {type: 'editClient'; client: Client} | null

3. Click-Outside Detection
   useEffect for mousedown listener with ref checking

4. Filtered & Sorted Lists
   useMemo with dependencies for performance

5. Form Validation
   Required field checks + error state display

6. Derived State Computation
   useMemo for ALL_USERS and selectedDeal

7. Callback Props Pattern
   24 handlers passed down to subcomponents

8. Timestamp-based IDs
   prefix-Date.now() for uniqueness
